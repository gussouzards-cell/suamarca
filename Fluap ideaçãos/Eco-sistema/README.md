# Fluap – Core de Benefícios

Web app de infraestrutura de benefícios com design minimalista e estilo enterprise.

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Estrutura

- **Sidebar**: Menu lateral fixo com navegação
- **Header**: Cabeçalho com logo placeholder e informações do produto
- **Páginas**: Estrutura base para cada seção do sistema

## Design

- Cores: Preto (#000000), Branco (#FFFFFF), Cinza (#171717)
- Tons de cinza via opacidade
- Bordas arredondadas (10px)
- Layout premium e minimalista
